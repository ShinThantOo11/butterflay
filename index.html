Butterfly Program ဆိုတာဘာလဲ?

Butterfly Program ဆိုတာကဘာလဲဆိုတာ စတင်မိတ်ဆက်ပေးပါမယ်။ ဂျပန်အိုင်တီကုမ္ပဏီကို မဝင်ရောက်ခင် IT Company Job Readyness ဖြစ်အောင်လေ့ကျင့်ပေးတဲ့ ပရိုဂရမ်တစ်ခုပါ။ဂျပန်နိုင်ငံရှိ Bliss Stock IT company မှ Software Engineer career ဖြင့်ဂျပန်နိုင်ငံတွင်လာရောက်အလုပ်လုပ်ကိုင်လိုတဲ့မြန်မာလူငယ်များအတွက် လုပ်ငန်းခွင်မဝင်ခင်မှာ Job readiness လုပ်ငန်းခွင်ဝင်ရောက်ရန်အသင့်အနေအထားဖြစ်အောင် လေ့ကျင့်သင်ကြားပေးမယ့်programဖြစ်ပါတယ်။ Programပထမအသုတ်အတွက် သင်တန်းသားဦးရေကတော့ ၁၀ဦးသာလက်ခံမှာဖြစ်ပါတယ်။

Butterfly Program မှာဘာတွေလုပ်ရမှာလဲ?

လုပ်ငန်းခွင်စဝင်ပြီဆိုတာနဲ့ must know သိကိုသိထားရမယ့်အချက်တွေရှိပါတယ်။ ကျောင်းမဟုတ်တော့တဲ့အတွက် အလုပ်ရှင်ဘက်ကနေ အလုပ်တခုချင်းစီကိုလိုက်သင်ပြပေးနေဖို့အချိန်မရှိသလို သင်ပြပေးမှာလည်းမဟုတ်တဲ့အတွက် ကိုယ့်ဘက်ကနေ ကြိုတင်လေ့လာထားဖို့လိုအပ်ပါတယ်။ ဥပမာအနေနဲ့ ရုံးတွင်းသုံး software များ tools များ၏အသုံးပြုပုံများ၊ Projectတခုဝင်လိုက်ပြီဆိုတာနဲ့ ပြင်ဆင်ရမည့်စာရွက်စာတမ်းများ၊ မိမိတစ်ဦးတည်းလုပ်နိုင်ရုံဖြင့်မလုံလောက်သဖြင့်အခြားသောအဖွဲ့ဝင်များနှင့်အတူပူးပေါင်းဆောင်ရွက်တတ်စေရန် စသည့်အကြောင်းအရာများကိုသင်ကြားပေးမှာဖြစ်ပါတယ်။

Butterfly Program ကဘယ်လောက်ကြာမြင့်မှာလဲဲ? ကန့်သတ်ချက်တွေက ဘာတွေလဲ?

Butterfly program ကကြာချိန်အားဖြင့်သုံးလကြာမှာဖြစ်ပြီး လျှောက်ထားသူတွေက language တစ်ခုခုနဲ့ project ရေးသားဖူးသူ၊ တစ်နည်းအားဖြင့် software development အခြေခံရှိသူဖြစ်ရပါမယ်။ ပြီးတော့ ဂျပန်ကုမ္ပဏီအလုပ်လုပ်ရမှာဖြစ်ပြီး Japan လူမျိုးတွေနဲ့ပြောဆိုဆက်ဆံရမှာဖြစ်တဲ့အတွက် Japanစာအဆင့် အနည်းဆုံး N4 အောင်မြင်ထားသူလည်းဖြစ်ရပါမယ်ခင်ဗျာ။ program ရဲ့အဓိကရည်ရွယ်ချက်က ဂျပန်IT company တွေမှာအလုပ်လုပ်တတ်အောင်သင်ပေးတာဖြစ်တဲ့အတွက် နာခံမှုအပြည့်နဲ့ လေ့လာသင်ယူလိုစိတ်ရှိရပါမယ်ခင်ဗျ။

သင့် အနာဂါတ်အတွက် ပြီးပြည့်စုံသော Butterfly OJT သင်တန်း

လူဘောင်အဖွဲ့အစည်းအတွင်း အလုပ်လုပ်တက်စေခြင်း

1.

ဘီးဇီးနစ်မန်နာ သင်ကြားခြင်းတွင် အစည်းအဝေးများနှင့် အင်တာဗျူးများ ကဲ့သို့သော ချိန်းဆိုမှုများတွင် အခြားသူများ၏ အချိန်ကိုလေးစားဖို့၊ အချိန်မှန်ရောက်ရှိစေရန်။ အချင်းချင်း ယဉ်ကျေးစွာ နှုတ်ဆက်တက်စေရန်။အခြားသူများ၏ ထင်မြင်ယူဆချက်များကို လေးစားပြီး ယဉ်ကျေးသောဘာသာစကားကို အသုံးပြုရန်။ လျှို့ဝှက်အချက်အလက်များနှင့် လျှို့ဝှက်အကြောင်းအရာများ ကိုလျှို့ဝှက်စွာထားတက်စေရန် အခြားသူများ၏ ကိုယ်ရေးကိုယ်တာကိစ္စများတွင် ပါဝင်ခြင်းမှ ရှောင်ကြဉ်ရန်။ အဖွဲ့လိုက်လုပ်ဆောင်ခြင်း၊ တင်ပြခြင်းနှင့် ဆွေးနွေးပြောဆိုခြင်း စွမ်းရည်များကို တိုးတက်စေရန် လေ့ကျင့်ပေးခြင်း။
အိုင်တီပညာရပ်ပိုင်းဆိုင်ရာအသိဉာဏ်တိုးပွားစေခြင်း။

2.

လက်တွေ့ ပရောဂျက်များတွင် ပါဝင်ခြင်းဖြင့်၊ လက်တွေ့ ပြဿနာဖြေရှင်းခြင်းနှင့် အမှန်တကယ်Software Developmentဖွံ့ဖြိုးတိုးတက်မှုဆိုင်ရာ အတွေ့အကြုံများကို သင်ရရှိနိုင်ပါသည်။ အဖွဲ့လိုက်လုပ်ဆောင်ခြင်းနှင့် Customerများနှင့် ဆက်သွယ်ခြင်းမှတစ်ဆင့် သင်၏လက်တွေ့စွမ်းရည ်များကို မြှင့်တင်နိုင်သည်။ အိုင်တီလုပ်ငန်းသည် အဆက်မပြတ်တိုးတက်နေသဖြင့်၊ နောက်ဆုံးပေါ်နည်းပညာများနှင့် ခေတ်ရေစီးကြောင်း များကို ခြေရာခံနိုင်ပြီး ကျွန်ုပ်တို့၏ Butterfly OJT ပရိုဂရမ်တွင် ပါဝင်ခြင်းဖြင့် လက်ရှိစွမ်းရည် ကိုယ်ပိုင်စွမ်းရည်ကို မြှင့်တင်နိုင်ပါသည်။
ဝန်ထမ်းသစ်လေ့ကျင့်ရေးဗီဒီယိုများမှ သင်ယူလေ့လာနိုင်ခြင်း။

3.

ယခင် Butterfly OJT Program အချိန်၌တိုက်ရိုက်သင်ကြားခဲ့သော၊ သင်ခန်းစာများမှ အမျိုးအစားအလိုက် အချိန်မရွေး ပြန်လှယ်လေ့လာနိုင်မည်။ အပတ်စဉ် Butterfly OJT Program များတွင် တော့သက်ဆိုင်ရာ ပရောဂျတ်များအလိုက် ကွဲပြားသည်များကိုတော့ Senior Member များမှ ထပ်ဆောင်းရှင်းလင်းပြသပါမည်။
မိမိစွမ်းရည်ကိုအကဲဖြတ်နိုင်ခြင်း

Butterfly OJT Program တွင်ပါဝင်သူများ၏ စွမ်းဆောင်ရည်ကို ပုံမှန်အကဲဖြတ်ပြီး ကျွန်ုပ်တို့၏ ကုမ္ပဏီအတွက် သင့်လျော်ပါက ၎င်းတို့အားလုပ်ငန်းဆောင်ရွက်မှုစွမ်းရည်၊ နည်းပညာဆိုင်ရာ ကျွမ်းကျင်မှု၊ ဆက်သွယ်ရေးစွမ်းရည်၊ ပြဿနာဖြေရှင်းနိုင်မှု၊ အလုပ်နှင့်ပတ်သတ်‌သော အကြောင်းကြားနိုင်မှု၊ အရည်အသွေးစသည်တို့အပေါ် အခြေခံအကဲဖြတ်ပါသည်။

Butterfly Program ကို ဘယ်လို JOIN ရမှာလဲ？

Butterfly Program ကို စိတ်ဝင်စားလို့ Joinချင်တယ်ဆိုရင် ကိုယ့်ရဲ့ CVအချက်အလက်တွေဖြည့်ထားသောဖောင်နှင့်တကွ info@bliss-stock.comသို့ သတ်မှတ်ထားသော Deadline မတိုင်ခင် အီးမေလ်ပေးပို့ပေးပါ။ ရောက်ရှိလာသော CVတွေထဲကမှ Coding Test ဖြေခွင့်ရသူတွေကိုရွေးချယ်ပြီး အီးမေလ်မှတဆင့်ပြန်လှယ်ဆက်သွယ်ပေးပါမယ်။ Coding Testမှာတော့ မိမိကြိုက်နှစ်သက်ရာ Programming Languageတခုခုနှင့်ဖြေဆိုနိုင်အောင်စီစဉ်ပေးထားပါတယ်။ Online မှတဆင့်ဖြေရမှာပါ။ Coding Test ပြီးရင်တော့ လူတွေ့အင်တာဗျူးကို Onlineကနေ ဖြေဆိုရပါမယ်။ အင်တာဗျူးမယ့်ရက်ကတော့ Coding Test ဖြေပီ တပတ်ခန့်အကြာမှာ နှစ်ရက်ခွဲပြီး ပြုလုပ်မာဖစ်ပြီး နောက်ဆုံးတက်ခွင့်ရတဲ့ စာရင်းကိုတော့ လူတွေ့အင်တာဗျူးဖြေအပြီး နှစ်ရက်ခန့်အကြာမှာ ထုတ်ပြန်ပေးသွားမှာပါ
